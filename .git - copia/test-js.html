Prueba de Javascript

si el HTML proporciona la estructura del documento,el CSS el estilo vusual entonces el JS proporsiona la interactividad de la web con el usuario.

Con JS podemos hacer que la pagina reacciones ante las distintas acciones del usuario,hacer clic,pasar el raton sobre un objeto en concreto, hacer scroll,usar el teclado

Al igual que el HTML nos provee de la etiqueta style para insertar el css.tenemos la etiqueta script para escribir JS ,el codigo tambien puede incorporase como atributo de los elementos (como sucede con el atributo style) por lo que su uso dependera del criterio del desarrollador.
